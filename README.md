# Stress Management Framework

## Overview

A proactive stress management system that uses behavioural and lifestyle analytics to identify stress patterns and provide insights for improving well-being.

## Technologies Used

* Python
* SQL
* Pandas
* Flask
* OpenCV

## Features

* Stress pattern analysis
* Behavioural data tracking
* Lifestyle analytics
* Data visualisation
* User-friendly interface
